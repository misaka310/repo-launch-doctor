No repository under the maintainer's control currently has all required evidence: a public repository, immutable before and after commits, reproducible Doctor reports for both commits, and a demonstrated link from finding to fix. ## Admission requirements for future cases

Each case must record the public repository URL, before and after SHA, both generated reports, the finding, the concrete change, and any false positive or problem not detected by the tool. A case becomes countable only when all evidence remains reproducible from the two commits.
